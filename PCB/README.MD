# PCB

KiCad design files for the glider's custom boards.

## Subsystems

- **P&R/** — Pitch & Roll board. Drives the pitch and roll actuators; also connects to the Raspberry Pi.
- **VBD/** — Variable Buoyancy Device board. Drives the VBD pump/valve subsystem.

Each folder contains the KiCad schematic (`.kicad_sch`) and PCB layout (`.kicad_pcb`); `P&R/` also includes the project file (`.kicad_pro`).
