# Battery Board

KiCad design files for the glider's battery boards.

## Subsystems

- **P&R/** — Pitch & Roll battery board. Powers the P&R subsystem; A and B variants.
- **VBD/** — Variable Buoyancy Device battery board. Powers the VBD subsystem; A and B variants.

Each folder contains the KiCad schematic (`.kicad_sch`), PCB layout (`.kicad_pcb`), and project file (`.kicad_pro`) for both A and B revisions.
